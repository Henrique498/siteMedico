# siteMedico

Site informativo desenvolvido como **Projeto Integrador (PI) para o curso Técnico em Medicina** do **Senac Taboão da Serra**. Construído com HTML e CSS puros, com foco em organização de conteúdo e navegação clara entre páginas.

---

## páginas

```
index.html        →  página principal
localizacao.html  →  localização da clínica/unidade
niveis.html       →  níveis de atendimento
orientacoes.html  →  orientações ao paciente
```

## como executar

```bash
# Clone o repositório
git clone https://github.com/Henrique498/siteMedico.git

# Acesse a pasta
cd siteMedico

# Abra o arquivo no navegador
start index.html
```

> Não requer instalação de dependências — basta abrir o `index.html` diretamente no navegador.
